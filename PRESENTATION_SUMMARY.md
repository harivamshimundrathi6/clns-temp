## Live Demo Flow (5-7 minutes)

### 1. Show Client Journey (2 minutes)
**"Let me show you how a client uses CLNS:"**

1. **Register** → "Client creates account in 30 seconds"
2. **Find Lawyer** → "Browse verified advocates with search"
3. **Book Consultation** → "One click creates a real case"
4. **Track Case** → "See status, hearings, communicate with lawyer"

**Key Point**: "Everything is real - cases are in the database, advocates see them immediately."

### 2. Show Student Journey (2 minutes)

1. **Upload Resume** → "Students upload their CV"

3. **Apply** → "One-click application with cover note"
4. **Get Hired** → "Advocate reviews resume, accepts application"

**Key Point**: "Advocates can see student resumes before hiring - making better decisions."

### 3. Show Advocate Dashboard (2 minutes)

3. **Manage Cases** → "Create cases, schedule hearings, update status"
4. **Contact Clients** → "Direct communication"

**Key Point**: "Everything advocates need in one place - no more switching between systems."

### 4. Show Admin Controls (1 minute)

3. **System Logs** → "Track all platform activity"

**Key Point**: "Complete control and visibility over the platform."
